# 💌 Forme – Diary-based Emotional Analysis Web App (Backend)

Forme는 사용자의 일기를 기록하고,  
그 내용을 바탕으로 감정을 분석하여 시각화하고,  
과거의 나와 “대화”할 수 있도록 설계된 웹 애플리케이션입니다.

본 프로젝트는 **백엔드 실습 과목**을 기반으로 진행된  
**Java Servlet & JSP 기반 백엔드 웹 프로젝트**입니다.

---

## 📌 Project Overview

- **Project Name**: Forme (For Me)  
- **Repository**: backend-forme  
- **Purpose**: Diary + Emotion Analysis + Self Reflection  
- **Type**: Backend Practice Project  
- **Framework**: Java Servlet / JSP  
- **Database**: MySQL  
- **Server**: Apache Tomcat  

Forme는 사용자가 매일의 감정을 일기로 남기면,  
해당 텍스트를 기반으로 감정 요소를 분석하고 시각적으로 표현하는 기능을 제공합니다.

## ✨ Main Features

- ✅ 회원가입 / 로그인 / 로그아웃  
- ✅ 일기 작성 / 수정 / 삭제 / 목록 조회 (CRUD)  
- ✅ 감정 분석 기능  
- ✅ 과거의 나와 대화하기 (채팅 형식)  
- ✅ 감정 시각화 (캐릭터, 색상, 퍼센트 등)  
- ✅ 정신건강 관련 기관 추천  
- ✅ 관리자 페이지 기능

## 🎯 Project Goal

이 프로젝트의 목표는 단순한 일기장이 아닌,  
**자신의 감정을 이해하고 정리할 수 있는 디지털 공간을 만드는 것**입니다.

> "기록은 기억을 만들고, 감정은 나를 만든다."  
Forme는 그 과정을 함께하는 웹 서비스입니다.
